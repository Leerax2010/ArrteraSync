установить библеотеку keyboard.
Для задавания параметров синхронизации заходить в файл SaveDirectories.py, в нем есть инструкция для задавания параметров.
Для настроек заходить в Settings.py, time-время указывать в секундах, show_info-показывать информацию о сохранение, True включить, False выключить.
И для запуска использовать файл ArterraSync.py и начнется синхронизация, а для выхода нажимать CTRL + Q, и установить библеотеку keyboard не забудь.
